# characters.py

## 역할
인물 카드 메타데이터의 단일 진실 소스다.

## 핵심 데이터
- `CharacterDef`: 카드 번호, 속성, 우선순위, 텍스트, 시작 액티브 여부를 담는다.
- `CHARACTERS`: 이름 -> 정의 맵.
- `CARD_TO_NAMES`: 카드 번호 -> 양면 인물 이름 쌍.
- `STARTING_ACTIVE_BY_CARD`: 고정 시작 액티브 면 템플릿.
- `randomized_active_by_card(rng)`: 각 카드의 시작 액티브 면을 양면 중 랜덤으로 뽑는 helper.

## 설계 의도
- 실제 능력 로직은 `engine.py`에 있지만, 공개 텍스트와 카드 구조는 여기서만 관리한다.
- 카드 flip, 드래프트, 설명 출력, 문서 생성이 모두 이 정보를 공유한다.
- 최신 기본 규칙에서는 실제 게임 시작 시 8개 카드의 시작 액티브 면을 이 템플릿에 고정하지 않고 `randomized_active_by_card(rng)`로 랜덤 세팅한다.

- 수정 규칙: 대응 소스 수정 시 이 문서도 함께 갱신한다.


## 최신 규칙 보정
- `사기꾼`은 토지 인수 시 그 칸에 적립된 승점 코인도 함께 인수한다.

- 교리 연구관/감독관: 턴 시작 액티브로 자신 또는 팀원의 짐 카드 1장을 제거하고, 턴 종료 후 징표를 이동한다. 현재 기본 자유대전 규칙에는 team_id가 없어 기본적으로 자기 짐을 우선 정리한다.
