# 2026-04-16 Rule Alignment Checklist

## 수정 대상

- `Public Event` 시작 흐름
  - 시작 손패 지급 중 즉시 `hidden_trick_card` 프롬프트가 열리던 흐름을 제거
  - 라운드의 `weather -> draft -> hidden trick slot sync -> round order` 흐름으로 정리

- `날씨`
  - `맑고 포근한 하루`: 각 플레이어가 회복할 주사위 카드를 직접 선택하도록 수정
  - `운수 좋은 날`: `S` 칸 도착 시 운수를 2장 연속 해결하도록 수정

- `운수`
  - `수상한 음료`: 1개 주사위만 굴리도록 수정
  - `남 좋은 일`: `무뢰` 제외 처리 추가
  - `청약 당첨`, `땅 도둑`, `기부천사`, `거절할 수 없는 거래`, `경건한 징표`
    - 선택형 타일 대상을 직접 고르도록 수정
  - `돼지 꿈`
    - 무작위 회복이 아니라 사용한 주사위 카드 중 2장을 직접 선택해 회복하도록 수정

- 브라우저에서 선택형 운수 프롬프트가 자연스럽게 노출되는지 재검증 필요
